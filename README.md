# Resume Website

Simple multi-page personal resume and portfolio website built with HTML and CSS.

## Pages

- `index.html` for the landing page
- `about.html` for background details
- `experience.html` for work and project experience
- `projects.html` for project highlights
- `contact.html` for contact details

## Purpose

This repository captures an early personal portfolio implementation focused on clean navigation and a straightforward resume-style presentation.
